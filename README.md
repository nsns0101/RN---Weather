# NPM, Android Studio 설치
- NPM : https://nodejs.org/ko/download/

# Expo-cli 설치
- npm install -g expo-cli

# Expo 프로젝트 생성
- expo init ProjectName

# 프로젝트 실행
- 1. AndroidStudio 에뮬레이터 실행
- 2. npm start, expo start로 프로젝트 실행(package.json 참고)
